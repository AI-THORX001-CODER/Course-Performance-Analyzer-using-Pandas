# Course-Performance-Analyzer-using-Pandas
A beginner Pandas project that analyzes course performance by calculating average marks, total marks, and student enrollment statistics to generate course-wise academic reports
🚀 Features
Create and manage student data using Pandas DataFrame
Group students by course
Calculate average marks for each course
Calculate total marks for each course
Count enrolled students per course
Generate course-wise performance reports
🛠️ Technologies Used
Python 3
Pandas
